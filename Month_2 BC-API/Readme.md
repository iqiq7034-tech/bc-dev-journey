- JSON handling — JsonObject, JsonArray, JsonToken, JsonValue
- BC API Pages — PageType = API, OData queries
- HttpClient in AL — outbound calls with full error handling
- OAuth 2.0 — token fetch, Authorization header
- Webhooks and Job Queue for scheduled sync
- Power Automate + BC API connection
